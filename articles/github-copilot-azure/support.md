# Getting support for GitHub Copilot for Azure

If you experience problems when using GitHub Copilot for Azure, the product team can help. However, the type of problem you have determines _where_ you find support. Since GitHub Copilot for Azure is one of many extensions to GitHub Copilot, this article provides some guidance on where to look for support for your problem.

::: zone pivot="visual-studio-code"

## GitHub Copilot support

Get support for GitHub Copilot when you experience the following problems:

- Copilot or GitHub authentication problems
- Problems with a large language model (LLM)

To get support, [create a new issue on the GitHub Copilot repo](https://github.com/microsoft/vscode-copilot-release/issues).

## Azure MCP Server

GitHub Copilot for Azure supplements the general knowledge of a foundational large language model (LLM) like GPT-5 and Claude Sonnet 4 with tool calling by using the **Azure Model Context Protocol (MCP) Server**. This protocol enables interaction with Azure services, systems, and Azure Resource Graph to carry out specific tasks on your behalf.

Get support for Azure MCP Server when you experience the following problems:

- Azure authentication problems
- Incorrect tool calling (the LLM calls the wrong tools)
- Tool performance (calls to the tools take too long, or return unusable results)

To get support, [create a new issue on the GitHub Copilot for Azure repo](https://aka.ms/azmcp).

## GitHub Copilot for Azure support in Visual Studio Code

For support with GitHub Copilot for Azure, see the following list of issues:

- Installation problems

To get support, [create a new issue on the GitHub Copilot for Azure repo](https://github.com/microsoft/GitHub-Copilot-for-Azure/issues).

::: zone-end

::: zone pivot="visual-studio-2022"

## GitHub Copilot for Azure support in Visual Studio 2022

For support with GitHub Copilot for Azure in Visual Studio 2022, see the following list of issues:

- Installation problems
- Azure authentication problems
- Incorrect tool calling (the LLM calls the wrong tools)
- Tool performance (calls to the tools take too long, or return unusable results)

Use Visual Studio 2022's **Report a Problem** menu item. On the menu bar, choose **Help** > **Send Feedback** > **Report a Problem**. This action opens a web browser to the **Report a problem** page on the Developer Community site.
::: zone-end

::: zone pivot="visual-studio-2026"

## GitHub Copilot for Azure support in Visual Studio 2026

For support with GitHub Copilot for Azure in Visual Studio 2026, seek help when you experience the following problems:

- Installation problems
- Azure authentication problems
- Incorrect tool calling (the LLM calls the wrong tools)
- Tool performance (calls to the tools take too long, or return unusable results)

Use Visual Studio 2026's **Report a Problem** menu item. On the menu bar, choose **Help** > **Send Feedback** > **Report a Problem**. This action opens a web browser to the **Report a problem** page on the Developer Community site.

::: zone-end
