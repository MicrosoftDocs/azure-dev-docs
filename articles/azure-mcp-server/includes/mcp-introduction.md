The Azure MCP Server enables AI agents and assistants to interact with Azure resources through natural language commands using the [Model Context Protocol (MCP)](https://modelcontextprotocol.io/). Instead of manually navigating portals or writing scripts, you can describe what you want to accomplish, and the AI assistant uses the Azure MCP Server tools to perform the actions.
