### MCP client requirements

You need an AI assistant or development environment that supports the Model Context Protocol. Choose one:

**AI-powered code editors**:
- [Cursor](../get-started/tools/cursor.md)
- [Cline](../get-started/tools/cline.md)
- [IntelliJ IDEA with MCP support](../get-started/tools/jet-brains.md)
- [Visual Studio](../get-started/tools/visual-studio.md)
- [Visual Studio Code](../get-started/tools/visual-studio-code.md)
- [Windsurf](../get-started/tools/windsurf.md)

**Programmatic integration**:
- [.NET with MCP SDK](../get-started/languages/dotnet.md)
- [Python with MCP SDK](../get-started/languages/python.md)

For complete setup instructions, see [Get started with Azure MCP Server](../get-started.md).
