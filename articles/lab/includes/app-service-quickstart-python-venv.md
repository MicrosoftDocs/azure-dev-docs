# [Bash](#tab/bash)

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

```powershell
py -3 -m venv .venv
.venv\scripts\activate
pip install -r requirements.txt
```

```cmd
py -3 -m venv .venv
.venv\scripts\activate
pip install -r requirements.txt
```
